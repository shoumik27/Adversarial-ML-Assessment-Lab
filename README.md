# Adversarial ML Assessment Lab

## Overview
Real-world evaluation of CNN robustness against adversarial attacks on CIFAR-10.

## Results
- **Baseline Accuracy**: 75%
- **FGSM Attack Success**: 93.75% at ε=0.3
- **Adversarial Training Improvement**: 22.5%
